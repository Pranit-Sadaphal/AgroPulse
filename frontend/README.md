# AgroPulse Frontend

React + Vite frontend for the AgroPulse crop yield prediction system.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Start development server:
```bash
npm run dev
```

3. Build for production:
```bash
npm run build
```

## Environment Variables

Create a `.env` file:
```
VITE_API_URL=http://localhost:8000
```

## Features

- Landing page
- Prediction dashboard
- Insights page
- Dark/light mode toggle
- Responsive design
